AR-7778_Electronic_Calculator_freqence_analyzer
===

## 簡介
* 佳靈通電子計算機按鍵音頻分析

## code 功能
### calculator_freq_analyzer
* real time 分析計算機的按鍵是哪一個
### cut_freq
* 分析一整個 wav 檔的按鍵有哪些
---
### average_freq
* 求出計算機中每個音 750 ~ 2000 Hz 的平均
### top_freq
* 求出計算機中每個音 stft 後最高頻率的平均
---
### generate_freq
* 印出 wav 檔的波形圖及 stft 後的頻譜圖
### top_five_freq
* 印出計算機中每個音 stft 後前五大的頻率圖

## 聲音檔
### calculator_file
* 計算機取樣檔
### harmonica_file
* 口琴取樣檔
